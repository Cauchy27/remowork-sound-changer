# Remowork Sound Changer - インストールガイド

Remoworkの着信音・通知音をカスタマイズするChrome拡張機能です。

## インストール手順

### 1. 拡張機能をダウンロード

`remowork-sound-changer` フォルダをローカルに保存します。

### 2. Chromeの拡張機能ページを開く

1. Chromeを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. または、メニュー → 拡張機能 → 拡張機能を管理

### 3. デベロッパーモードを有効化

ページ右上の「デベロッパーモード」トグルをONにします。

### 4. 拡張機能を読み込む

1. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
2. `remowork-sound-changer` フォルダを選択
3. 「選択」をクリック

### 5. インストール完了

ツールバーにスピーカーアイコンが表示されれば成功です。

---

## 使い方

### 設定画面を開く

ツールバーの拡張機能アイコンをクリックすると設定画面が開きます。

### 音声を変更する

1. 変更したい音声の行を見つける
2. ドロップダウンで「カスタム」を選択
3. アップロードボタン（↑）をクリック
4. 音声ファイル（MP3, WAV, OGGなど）を選択

### 音声をプレビュー

再生ボタン（▶）をクリックすると、設定した音声をプレビューできます。

### オリジナルに戻す

ドロップダウンで「オリジナル」を選択すると、Remoworkのデフォルト音声に戻ります。

### 拡張機能を無効化

右上のトグルスイッチをOFFにすると、音声の差し替えを一時的に無効化できます。

## 音声の種類

| 名前 | 説明 |
|------|------|
| 発信中（呼び出し音） | 相手を呼び出し中に鳴る音 |
| 着信音 | 電話がかかってきた時に鳴る音 |
| 発信音 | 発信ボタンを押した時に鳴る音 |
| 切断音 | 通話が終了・切断された時に鳴る音 |
| ドアチャイム | 内線着信時に鳴る音 |

## 使用例・アイデア

自分好みの音に変えて、仕事をもっと楽しく！

### 着信音のアイデア
- 猫の鳴き声「にゃー」
- 推しの声（ボイスサンプルなど）
- 好きなアニメのSE
- 妹の鼻歌（録音させてもらおう）
- レトロな黒電話の音

### 切断音のアイデア
- ファンファーレ（お疲れ様！）
- ゲームのレベルアップ音
- 「ミッション完了」的なSE
- 拍手の音
- コインゲット音

### 発信音のアイデア
- 冒険に出発するようなBGM
- 「いってきます」の声
- ロケット発射音

### 内線（ドアチャイム）のアイデア
- ピンポーン（リアルなチャイム）
- 「はーい」という声

## 録音機能

Remoworkサイト上に表示されるフローティングパネルから、通話を録音できます。

### 録音の使い方

1. Remoworkサイトを開くと、右上に録音パネルが表示されます
2. 赤い録音ボタン（●）をクリックして録音開始
3. 「タブを共有」ダイアログが表示されたら、**現在のタブを選択**
4. **「タブの音声を共有」にチェック**を入れる（重要！）
5. 通話が終わったら停止ボタン（■）をクリック
6. 録音履歴からダウンロード（↓）または再生（▶）

### 双方向録音の注意点

通話の**自分の声**と**相手の声**の両方を録音するには、以下の条件が必要です：

| 音声ソース | 取得方法 | 注意点 |
|-----------|----------|--------|
| 自分の声 | マイク権限 | ブラウザのマイク許可が必要 |
| 相手の声 | タブ音声共有 | 録音開始時に「タブの音声を共有」を有効にする |

> **重要**: 録音開始時のダイアログで「タブの音声を共有」にチェックを入れないと、相手の声が録音されません。

### 録音ファイルについて

- 形式: WebM（Opus コーデック）
- 多くの音声プレイヤーで再生可能
- MP3への変換が必要な場合は、オンライン変換ツールをご利用ください

### パネルの操作

- **ドラッグ**: ヘッダー部分をドラッグして位置を移動
- **最小化**: 右上の「−」ボタンでパネルを最小化
- **削除**: 録音履歴のゴミ箱アイコンで録音を削除

## トラブルシューティング

### 音声が変わらない

1. 拡張機能が有効になっているか確認（トグルがON）
2. ページをリロード（Ctrl+R / Cmd+R）
3. 拡張機能を再読み込み（chrome://extensions/ で更新ボタン）

### アップロードできない

- ファイルサイズが300MBを超えていないか確認
- 音声ファイル（MP3, WAV, OGG, M4A）か確認

### アイコンが表示されない

1. chrome://extensions/ で拡張機能が有効か確認
2. ツールバーの拡張機能アイコン（パズルピース）をクリック
3. 「Remowork Sound Changer」をピン留め

---

## アンインストール

1. `chrome://extensions/` を開く
2. 「Remowork Sound Changer」の「削除」をクリック
